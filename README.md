# Short Video Attention Analysis

This repository contains a small experimental project analyzing
the effect of short-form video consumption on attention span
and academic performance.

## Project Goal
To explore whether high exposure to short videos
impacts sustained attention and memory retention.

## Data Source
- Student survey (self-reported usage hours)
- Focus task performance test
- GPA comparison

## Planned Analysis
- Correlation between hours spent & GPA
- Focus duration comparison
- Basic statistical visualization

## Tools
- Python
- Pandas
- Matplotlib
